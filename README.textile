h2. ENV notes

* postgresql db required to match staging and production
* install with brew install postgresql

h3. postgresql tips

* cli = psql qwiqq_development
* list tables = \dt
* list databases = \l
* exit = \q

h3. Data

h3. Images

* run heroku config for AWS creds

h2. API notes

* return all ids as string using .try(:to_s)
* prefix id with model, eg user_id
* current user is a valid option for user_id or id for users collection
